# 📊 Churn Intelligence Dashboard (Power BI)

## 🎯 Project Overview
This project delivers a 2-page interactive Power BI dashboard designed to analyze customer churn behavior for a telecom company. The goal is to identify why customers leave (churn), uncover high-risk business areas, and provide data-driven recommendations to maximize revenue retention.

## 💡 Key Business Problems Solved
1. **Competitor Threat Analysis:** Discovered that competitors are the leading cause of customer loss. Specifically, **Fiber Optic** internet users show a staggering **40.72%** churn rate, indicating competitive pricing or quality gaps.
2. **Contract Risk Mitigation:** Identified that **Month-to-Month** contracts represent a major risk with a **45.84%** churn rate, compared to just 2.55% for long-term 2-year contracts.
3. **Behavioral Feature Impact:** Analyzed consumer trends showing that customers who adopt **Online Security** add-ons display significantly higher loyalty profiles (48.31% adoption in stable groups like DSL).

## 🛠️ Skills & Technologies Used
* **Data Transformation:** Power Query (Data cleaning, handling blanks, types formatting).
* **Analytical Modeling:** DAX measures created for core business metrics (e.g., `Avg Tenure`, `Total Refunds`, `Churn Rate`, `Extra Revenue`).
* **UI/UX Design:** Built with a clean, modern grid layout using intuitive custom "card" visuals, consistent corporate color palettes, and balanced white space for quick data scannability.

## 🚀 How to Interact with the Project
1. Download the `.pbix` file from this repository.
2. Open it using **Power BI Desktop**.
3. Use the top interactive **Slicers** (Filters) like `Gender`, `City`, or `Churn Reason` to dynamically slice and deep-dive into the live metrics.
